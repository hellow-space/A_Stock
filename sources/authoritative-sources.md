# 研究来源分级与引用规则

“可访问”不等于“权威”。研究时优先使用一级原始来源，二级研究与媒体只用于补充背景、发现线索和交叉验证。

## 一级：原始与法定来源

| 来源 | 入口 | 适用内容 |
|------|------|----------|
| 中国证监会 | https://www.csrc.gov.cn | 监管政策、行政处罚、制度文件 |
| 上海证券交易所 | https://www.sse.com.cn | 沪市公司公告、交易规则、市场数据 |
| 深圳证券交易所 | https://www.szse.cn | 深市公司公告、交易规则、市场数据 |
| 北京证券交易所 | https://www.bse.cn | 北交所公司公告与规则 |
| 中国人民银行 | https://www.pbc.gov.cn | 货币政策、LPR、金融统计数据 |
| 国家统计局 | https://www.stats.gov.cn | GDP、CPI、PPI、PMI 等宏观数据 |
| 巨潮资讯 | https://www.cninfo.com.cn | 上市公司法定信息披露 |
| 公司投资者关系页面 | 各公司官网 | 财报、演示材料、投资者问答原文 |

公司基本面、重大事项和监管结论应尽量回到这一层核验。

## 二级：专业研究与数据工具

| 来源 | 入口 | 使用方式 |
|------|------|----------|
| Wind | https://www.wind.com.cn | 结构化数据与机构研究，注意授权和指标口径 |
| 东方财富 Choice | https://choice.eastmoney.com | 行情与财务数据，关键数字需交叉核验 |
| 同花顺 iFinD | https://www.ifind.com | 行情、行业和财务数据，记录查询日期 |
| Bloomberg | https://www.bloomberg.com | 全球市场与跨资产数据 |
| 中金公司 | https://www.cicc.com | 宏观、策略和行业研究 |
| 中信证券 | https://www.citics.com | 策略与行业研究 |
| 申万宏源研究 | https://www.swsresearch.com | 行业分类和行业研究 |

这一层可能需要订阅，也可能存在预测偏差。引用时记录报告名称、作者、发布日期和页码，不把评级直接当作事实。

## 三级：新闻与市场线索

| 来源 | 入口 | 适用方式 |
|------|------|----------|
| 财新 | https://www.caixin.com | 新闻调查与政策背景，重要结论回查原文 |
| 华尔街见闻 | https://wallstreetcn.com | 快讯和市场线索，不作为唯一证据 |
| 东方财富网 | https://www.eastmoney.com | 行情入口与资讯聚合 |
| 雪球 | https://xueqiu.com | 观察市场观点和情绪，不作为事实来源 |

## 引用最低要求

每项关键数据至少记录：

1. 数据名称与统计口径。
2. 数据截止日期或报告期。
3. 发布机构和原始链接。
4. 获取日期；动态网页尤其需要记录。

出现冲突时，优先级通常为：法定披露或监管文件、机构原始数据、专业数据库、研究报告、新闻媒体、社区观点。
