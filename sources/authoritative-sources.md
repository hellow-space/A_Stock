# 权威信息来源汇总

## 一、监管机构（政策、法规、市场数据）

| 机构 | 网址 | 核心信息 |
|------|------|----------|
| 中国证监会 (CSRC) | https://www.csrc.gov.cn | 政策法规、IPO审核、信息披露、行政处罚 |
| 上海证券交易所 | https://www.sse.com.cn | 上证指数、上市公司公告、科创板 |
| 深圳证券交易所 | https://www.szse.cn | 深证/创业板指数、交易数据 |
| 北京证券交易所 | https://www.bse.cn | 北交所公司信息 |
| 中国人民银行 | http://www.pbc.gov.cn | 货币政策、LPR、社融数据 |
| 国家统计局 | https://www.stats.gov.cn | GDP、CPI、PPI、PMI |

## 六、按研究目的快速选择

| 研究目的 | 首选来源 |
|----------|----------|
| 宏观政策/监管动态 | 证监会官网、央行官网 |

| 行业深度研究 | 申万/中金/中信行业研报 |
| 全球市场/跨资产 | 高盛/摩根大通全球策略、Bloomberg |
| 量化数据 | Wind、Choice、券商金工研报 |
| 实时市场热点 | 东方财富、华尔街见闻、财新 |
